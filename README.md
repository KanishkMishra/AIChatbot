# AI Assistant

An AI-powered chatbot designed to answer any queries about the topic of choice.

## Features
- Natural language conversation (Gemini API)
- Web-based chat interface
- Scalable backend with Flask

## Tech Stack
- Python (Flask)
- Google Gemini API
- Groq API
- HTML/CSS/JavaScript

## Setup
1. Clone repo
2. Add `.env` file with API key
3. Enter vertual environemnt with python 3.13.12 `venv\Scripts\activate`
4. Run `python app.py`
